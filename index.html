🎮 Chrome-Dino 🎮

Google Chrome's Dino game is adapted to HTML so you can play it on every device by going to "chrome://dino" except for blocked ones.

This is a fix!

❗DISCLAIMER❗

This was tested on a Google Chromebook (OS Version: 84.0.4147.136) ✔️

This was tested on a Google Chromebook (OS Version: 84.0.4147.136) managed by an organization ✔️

This was tested on a Google Chromebook (OS Version: 84.0.4147.136) in a managed network ✔️

This was tested on Google Chrome (Version 111.0.5563.64 64BIT) ✔️

This wasn't tested on Bing, but should work as this browser supports HTML and JS execution ⚠️

This wasn't tested on Firefox, but should work as this browser supports HTML and JS execution ⚠️

⏯️ Playing ⏯️

That's kinda it! You use the space bar to play (press the space bar to jump) or you can use the arrow keys. (Up: Jump) Don't forget to let me know if there are any other issues.

If you are not redirected in five seconds, click here.
